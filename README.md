# 🎉 Website Ucapan Ulang Tahun untuk Pacar

Website ucapan ulang tahun yang indah dan interaktif untuk pacar tercinta! ✨

## Fitur

- 🎈 **Loading Screen** dengan animasi hati berdenyut
- 💌 **Surat Cinta** dalam amplop yang bisa dibuka
- 📷 **Galeri Foto** untuk kenangan bersama
- 🎂 **Kue Ulang Tahun** dengan lilin yang bisa ditiup
- 🎊 **Efek Confetti** yang meriah
- 💕 **Animasi Hati** yang mengikuti gerakan mouse
- 📱 **Responsive Design** untuk semua device

## Cara Menggunakan

1. Buka file `index.html` di browser
2. Klik layar loading untuk memulai
3. Ikuti alur cerita yang telah dibuat
4. Tiup lilin dan lihat harapan spesial!

## Kustomisasi

### Mengganti Foto
Edit bagian `.photo-placeholder` di `index.html` dan ganti dengan tag `<img>`:
```html
<img src="foto-pacar-kamu.jpg" alt="Foto Pacar" style="width: 100%; height: 100%; object-fit: cover; border-radius: 10px;">
```

### Mengganti Pesan
Edit teks di bagian `.message-content` sesuai dengan pesan pribadi kamu.

### Mengganti Nama
Cari teks "Nama Kamu" dan ganti dengan nama kamu.

### Menambahkan Musik
Uncomment bagian audio di `script.js` dan tambahkan file musik kamu.

## Struktur File

```
├── index.html      # Halaman utama
├── styles.css      # Styling dan animasi
├── script.js       # Interaktivitas dan efek
└── README.md       # Dokumentasi
```

## Teknologi

- HTML5
- CSS3 (Animations, Gradients, Flexbox, Grid)
- JavaScript (ES6+)
- Google Fonts (Dancing Script, Poppins)

## Tips

- Gunakan foto berkualitas tinggi untuk hasil terbaik
- Tulis pesan dari hati untuk membuatnya lebih personal
- Test di browser modern (Chrome, Firefox, Safari, Edge)
- Bisa di-hosting gratis di GitHub Pages atau Netlify
